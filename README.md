# Jogo - TCC

Este repositório é dedicado ao versionamento dos arquivos (de arte e de protótipo) do projeto ligado ao meu TCC "Desenvolvimento de jogo analógico educacional para a área de computação voltado a crianças e adolescentes".

## Estrutura do projeto

Na pasta "Root", você encontrará este ReadMe e o arquivo `tcc-isadora-vX.pcio`, o qual representa um protótipo digital a ser testado no site [playingcards.io/](https://playingcards.io/).

Na pasta "Componentes", você encontrará os diferentes componentes utilizados no jogo, como a peça do jogador, os tokens de Contador de Programa e Erro, o tabuleiro/playmat de jogo. Além disso, as cartas de Ação, de Receita e os tokens de Ingredientes se encontram nessa pasta.

Na pasta "PNPTool", você encontrará os arquivos base para o programa [PNPTool](https://boardgamegeek.com/thread/2490834/tool-for-help-with-your-game-prototype-tabletop-pn), o qual possibilita a criação de arquivos imrpimíveis.

Na pasta "Impressao" estão os arquivos PDF para impressão criados no PNPTool e outros softwares de edição de imagem.

## Desenvolvimento

A versão atual desse projeto foi desenvolvida usando a biblioteca de arte do Canva Pro e outros recursos CC0 encontrados na Web.

Algumas das artes utilizadas foram modificadas para melhor se adequarem ao jogo.

As atribuições corretas estão sendo escritas.
